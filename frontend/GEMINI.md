# Frontend Mandates

## Package Management
- **Primary Package Manager:** Always use **pnpm** for installing, adding, or removing packages.
- **Script Execution:** **npm** may be used to run scripts defined in `package.json` (e.g., `npm run build`, `npm run serve`).
- **Standard Library:** Adhere to the mandates in the root `GEMINI.md` regarding the use of `standard-ts-lib`.
